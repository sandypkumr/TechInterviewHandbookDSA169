## Problem Statement

Design an algorithm to encode a list of strings to a string. The encoded string is then sent over the network and is
decoded back to the original list of strings.

Please implement encode and decode


## Examples

Example:
    Input: ["Hello", "World", "this is a test"]
    codec = Solution()
    encoded_string = codec.encode(["Hello", "World", "this is a test"])
    print(encoded_string)
    Output: "5#Hello5#World14#this is a test"
    decoded_string = codec.decode(encoded_string)
    print(decoded_string)
    Output: ["Hello", "World", "this is a test"]


## Solution

```python
from typing import List


class Solution:
    def encode(self, strs: List[str]) -> str:
        return ''.join(str(len(s)) + '#' + s for s in strs)

    def decode(self, s: str) -> List[str]:
        res = []
        i = 0
        while i < len(s):
            j = i
            while s[j] != '#':
                j += 1
            length = int(s[i:j])
            res.append(s[j + 1:j + 1 + length])
            i = j + 1 + length
        return res
```


## Explanation

### Approach: Length-Prefixed Encoding

Each string is encoded as:

`<length>#<content>`

Example:
- `"Hello"` -> `"5#Hello"`

Concatenate all encoded pieces into one stream.

### Why This is Unambiguous

Delimiter `#` only separates length from content. Content can contain any characters, including `#`, because decoder reads exact number of characters based on parsed length.

### Decode Process

Scan with pointer `i`:
1. advance `j` until `s[j] == '#'` to parse length
2. read next `length` characters as one decoded string
3. move `i` to start of next token

### Code Core

```python
while i < len(s):
    j = i
    while s[j] != '#':
        j += 1
    length = int(s[i:j])
    res.append(s[j + 1:j + 1 + length])
    i = j + 1 + length
```

### Why This Works

- Length prefix gives exact boundaries for each string
- Encoding and decoding are inverse operations
- Runs in linear time over encoded data length

- **Time**: `O(n)`
- **Space**: `O(n)`
