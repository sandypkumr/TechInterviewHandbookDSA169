## Problem Statement

Implement pow(x, n), which calculates `x` raised to the power `n` (i.e., `x^n`).

 

**Example 1:**

```
Input: x = 2.00000, n = 10
Output: 1024.00000
```

**Example 2:**

```
Input: x = 2.10000, n = 3
Output: 9.26100
```

**Example 3:**

```
Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2^-2 = 1/2^2 = 1/4 = 0.25
```

 

**Constraints:**

- `-100.0 < x < 100.0`
- `-2^31 <= n <= 2^31-1`
- `n` is an integer.
- Either `x` is not zero or `n > 0`.
- `-10^4 <= x^n <= 10^4`

## Solution

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            x = 1 / x
            n = -n
        res = 1
        while n:
            if n & 1:
                res *= x
            x *= x
            n >>= 1
        return res
```


## Explanation

### Approach: Fast Exponentiation (Binary Exponentiation)

This solution uses **fast exponentiation** to compute x^n efficiently in O(log n) time using the binary representation of n.

### Intuition

**Divide and Conquer:** Instead of multiplying x by itself n times, use binary representation:
- x^13 = x^(1101₂) = x^8 × x^4 × x^1

We compute x^2, x^4, x^8, ... and combine based on binary digits of n.

### Algorithm

1. If n is negative, convert to positive (invert result)
2. Build powers: x^1, x^2, x^4, x^8, ...
3. For each '1' bit in n, include that power
4. Multiply all selected powers

### Why This Works

- **O(log n) Time:** We only need log n multiplications
- **Binary Representation:** Any exponent can be expressed as sum of powers of 2
- **Efficient:** Much faster than naive O(n) multiplication


## Complexity

- **Time**: `O(log n)`
- **Space**: `O(log n)`
