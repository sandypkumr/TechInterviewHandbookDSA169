## Problem Statement

Given the `root` of a binary tree and an integer `targetSum`, return _all **root-to-leaf** paths where the sum of the node values in the path equals _`targetSum`_. Each path should be returned as a list of the node **values**, not node references_.

A **root-to-leaf** path is a path starting from the root and ending at any leaf node. A **leaf** is a node with no children.

 

**Example 1:**

```
Input: root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
Output: [[5,4,11,2],[5,8,4,5]]
Explanation: There are two paths whose sum equals targetSum:
5 + 4 + 11 + 2 = 22
5 + 8 + 4 + 5 = 22
```

**Example 2:**

```
Input: root = [1,2,3], targetSum = 5
Output: []
```

**Example 3:**

```
Input: root = [1,2], targetSum = 0
Output: []
```

 

**Constraints:**

- The number of nodes in the tree is in the range `[0, 5000]`.
- `-1000 <= Node.val <= 1000`
- `-1000 <= targetSum <= 1000`

## Solution

```python
from typing import Optional, List


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> List[List[int]]:
        def dfs(node, path, target):
            if not node:
                return
            if not node.left and not node.right and node.val == target:
                res.append(path + [node.val])
                return
            dfs(node.left, path + [node.val], target - node.val)
            dfs(node.right, path + [node.val], target - node.val)
        res = []
        dfs(root, [], targetSum)
        return res
```


## Explanation

### Approach: DFS Backtracking on Root-to-Leaf Paths

We traverse all root-to-leaf paths while carrying:
- current `path`
- remaining `target`

At each node:
1. include node value in path
2. subtract node value from remaining target
3. if leaf and remaining target equals node value -> record path

Then recurse to left and right children.

### Step-by-Step Example

For target `22` in example tree:
- Path `5 -> 4 -> 11 -> 2` gives remaining `22-5-4-11-2 = 0` at leaf -> keep
- Path `5 -> 8 -> 4 -> 5` also reaches 0 at leaf -> keep

These become output paths.

### Code Breakdown


`path + [node.val]` creates independent path copies, avoiding shared-state bugs.

### Why This Works

- Enumerates every root-to-leaf path exactly once
- Filters only paths whose sum matches `targetSum`
- Time O(n) traversal with recursion stack O(h)
