## Problem Statement

Given a string `s` which consists of lowercase or uppercase letters, return the length of the **longest palindrome** that can be built with those letters.

Letters are **case sensitive**, for example, `"Aa"` is not considered a palindrome.

 

**Example 1:**

```
Input: s = "abccccdd"
Output: 7
Explanation: One longest palindrome that can be built is "dccaccd", whose length is 7.
```

**Example 2:**

```
Input: s = "a"
Output: 1
Explanation: The longest palindrome that can be built is "a", whose length is 1.
```

 

**Constraints:**

- `1 <= s.length <= 2000`
- `s` consists of lowercase **and/or** uppercase English letters only.

## Solution

```python
from collections import Counter

class Solution:
    def longestPalindrome(self, s: str) -> int:
        ans = 0
        for v in Counter(s).values():
            ans += v // 2 * 2
            if ans % 2 == 0 and v % 2 == 1:
                ans += 1
        return ans
```


## Explanation

### Approach: Greedy Algorithm

This solution uses a **Greedy** strategy to make locally optimal decisions that lead to the global optimum.

### Intuition

Every pair of identical characters can be placed symmetrically on the left and right sides of a palindrome. After using all possible pairs, at most one leftover odd-count character can sit in the center.

### Algorithm

Here's the step-by-step approach:

1. Count the frequency of each character.
2. For every frequency, add its largest even contribution to the answer.
3. If no center has been used yet and the current frequency is odd, add one extra character for the middle.
4. Return the total palindrome length.

### Code Breakdown

`v // 2 * 2` extracts the biggest even number not exceeding `v`, which is exactly how many copies of that character can be placed symmetrically. The condition `ans % 2 == 0 and v % 2 == 1` ensures that only one odd leftover is used as the center.

### Why This Works

- **Pair maximization:** Using all available pairs is always optimal because pairs contribute two positions each.
- **Single center rule:** A palindrome can contain at most one unpaired middle character.
- **Independent character counts:** The contribution of each character frequency can be computed separately and safely combined.
