## Problem Statement

A tree is an undirected graph in which any two vertices are connected by _exactly_ one path. In other words, any connected graph without simple cycles is a tree.

Given a tree of `n` nodes labelled from `0` to `n - 1`, and an array of `n - 1` `edges` where `edges[i] = [a_i, b_i]` indicates that there is an undirected edge between the two nodes `a_i` and `b_i` in the tree, you can choose any node of the tree as the root. When you select a node `x` as the root, the result tree has height `h`. Among all possible rooted trees, those with minimum height (i.e. `min(h)`) are called **minimum height trees** (MHTs).

Return _a list of all **MHTs'** root labels_. You can return the answer in **any order**.

The **height** of a rooted tree is the number of edges on the longest downward path between the root and a leaf.

 

**Example 1:**

```
Input: n = 4, edges = [[1,0],[1,2],[1,3]]
Output: [1]
Explanation: As shown, the height of the tree is 1 when the root is the node with label 1 which is the only MHT.
```

**Example 2:**

```
Input: n = 6, edges = [[3,0],[3,1],[3,2],[3,4],[5,4]]
Output: [3,4]
```

 

**Constraints:**

- `1 <= n <= 2 * 10^4`
- `edges.length == n - 1`
- `0 <= a_i, b_i < n`
- `a_i != b_i`
- All the pairs `(a_i, b_i)` are distinct.
- The given input is **guaranteed** to be a tree and there will be **no repeated** edges.

## Solution

```python
from typing import List


class Solution:
    def findMinHeightTrees(self, n: int, edges: List[List[int]]) -> List[int]:
        if n == 1:
            return [0]
        adj = [set() for _ in range(n)]
        for u, v in edges:
            adj[u].add(v)
            adj[v].add(u)
        leaves = [i for i in range(n) if len(adj[i]) == 1]
        while n > 2:
            n -= len(leaves)
            new_leaves = []
            for leaf in leaves:
                neighbor = adj[leaf].pop()
                adj[neighbor].remove(leaf)
                if len(adj[neighbor]) == 1:
                    new_leaves.append(neighbor)
            leaves = new_leaves
        return leaves
```


## Explanation

### Approach: Topological Leaf Pruning

The key idea is that minimum-height-tree roots must lie near the **center** of the tree. Instead of trying every node as a root, repeatedly remove the current leaves (nodes with degree 1). After trimming the outer layers, the last 1 or 2 nodes left are the centers, so they are exactly the valid MHT roots.

### Intuition

Leaves can never be optimal roots for minimum height unless the tree is extremely small. Every pruning round removes one layer from the outside, just like peeling an onion. The nodes that survive the longest are the most central ones.

### Algorithm

1. If `n == 1`, return `[0]`
2. Build an adjacency list using sets so edges can be removed efficiently
3. Collect all current leaves (nodes whose neighbor count is 1)
4. While more than 2 nodes remain:
   - remove all current leaves
   - update their neighbors
   - any neighbor whose degree becomes 1 becomes a new leaf
5. Return the remaining 1 or 2 nodes

### Code Breakdown

- `adj = [set() for _ in range(n)]`: stores each node's neighbors
- `leaves = [i for i in range(n) if len(adj[i]) == 1]`: starts with the outermost layer
- `n -= len(leaves)`: removes one full layer each round
- `neighbor = adj[leaf].pop()`: the only node connected to this leaf
- `adj[neighbor].remove(leaf)`: deletes the leaf edge
- `if len(adj[neighbor]) == 1`: this neighbor has become a new leaf

### Why This Works

- Every round removes nodes that are farthest from any possible center
- A tree has either one center or two centers
- Those centers are exactly the nodes left after all outer layers are stripped away
- Each edge is removed once, so the process is efficient
