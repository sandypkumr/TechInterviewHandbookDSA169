## Problem Statement

Given an integer array `nums`, find the subarray with the largest sum, and return _its sum_.

 

**Example 1:**

```
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.
```

**Example 2:**

```
Input: nums = [1]
Output: 1
Explanation: The subarray [1] has the largest sum 1.
```

**Example 3:**

```
Input: nums = [5,4,-1,7,8]
Output: 23
Explanation: The subarray [5,4,-1,7,8] has the largest sum 23.
```

 

**Constraints:**

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

 

**Follow up:** If you have figured out the `O(n)` solution, try coding another solution using the **divide and conquer** approach, which is more subtle.

## Solution

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        max_sum = nums[0]
        current_sum = 0
        for i in range(len(nums)):
            if current_sum < 0:
                current_sum = 0
            current_sum = current_sum + nums[i]
            max_sum = max(max_sum, current_sum)
        return max_sum
```


## Explanation

### Approach: Kadane's Algorithm

This solution uses **Kadane's Algorithm**, an elegant dynamic programming approach to find the maximum subarray sum in O(n) time and O(1) space.

### Intuition

**Key Idea:** At each position, decide whether to:
1. Extend the current subarray (if current_sum ≥ 0)
2. Start a new subarray (if `current_sum < 0`)

We track the maximum sum seen so far and update it as we process each element.

**Why Restart:** If the cumulative sum becomes negative, it only reduces future sums, so we discard it and start fresh.

### Step-by-Step Walkthrough

Example: `nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]`

```
Index:      0   1   2   3   4   5   6   7   8
Value:     -2   1  -3   4  -1   2   1  -5   4
current:   -2   1  -2   4   3   5   6   1   5
max_sum:   -2   1   1   4   4   5   6   6   6

At index 3: current_sum = 4, reset from previous -2 because it's negative
At index 6: current_sum = 6 (best answer so far)
```

### Code Breakdown

```python
current_sum = 0
for num in nums:
    if current_sum < 0:
        current_sum = 0  # Restart: discard negative prefix
    current_sum += num
    max_sum = max(max_sum, current_sum)
```

**Reset Logic:** If current_sum becomes negative, starting fresh is better than keeping a negative prefix.

### Why This Works

- **Optimal Substructure:** Maximum subarray ending at i depends on maximum ending at i-1
- **O(n) Time:** Single pass through array
- **O(1) Space:** Only tracking two variables
- **Correctness:** Handles all negative arrays and finds global maximum
