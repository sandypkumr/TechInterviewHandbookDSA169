## Problem Statement

Given a linked list, swap every two adjacent nodes and return its head. You must solve the problem without modifying the values in the list's nodes (i.e., only nodes themselves may be changed.)

 

**Example 1:**

**Input:** head = [1,2,3,4]

**Output:** [2,1,4,3]

**Explanation:**

**Example 2:**

**Input:** head = []

**Output:** []

**Example 3:**

**Input:** head = [1]

**Output:** [1]

**Example 4:**

**Input:** head = [1,2,3]

**Output:** [2,1,3]

 

**Constraints:**

- The number of nodes in the list is in the range `[0, 100]`.
- `0 <= Node.val <= 100`

## Solution

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(next=head)
        prev = dummy
        while prev.next and prev.next.next:
            first = prev.next
            second = prev.next.next
            prev.next, first.next, second.next = second, second.next, first
            prev = first
        return dummy.next
```


## Explanation

### Approach: Iterative Pairwise Pointer Rewiring

Use a dummy head and swap every adjacent pair by updating `next` pointers only.

For each pair:
- `first = prev.next`
- `second = first.next`

After swap, links become:

`prev -> second -> first -> (rest)`

Then move `prev` to `first` (new tail of swapped pair) and continue.

### Step-by-Step Example

Input: `1 -> 2 -> 3 -> 4`

- Swap `(1,2)` -> `2 -> 1 -> 3 -> 4`
- Move `prev` to `1`
- Swap `(3,4)` -> `2 -> 1 -> 4 -> 3`

Result: `2 -> 1 -> 4 -> 3`

### Code Breakdown

```python
dummy = ListNode(next=head)

```

Tuple assignment performs the three pointer updates cleanly.

### Why This Works

- Swaps are local and do not disturb already-processed prefix
- Dummy node handles head swaps uniformly
- Each node is visited constant times -> O(n) time, O(1) space
