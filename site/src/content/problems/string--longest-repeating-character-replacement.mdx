## Problem Statement

You are given a string `s` and an integer `k`. You can choose any character of the string and change it to any other uppercase English character. You can perform this operation at most `k` times.

Return _the length of the longest substring containing the same letter you can get after performing the above operations_.

 

**Example 1:**

```
Input: s = "ABAB", k = 2
Output: 4
Explanation: Replace the two 'A's with two 'B's or vice versa.
```

**Example 2:**

```
Input: s = "AABABBA", k = 1
Output: 4
Explanation: Replace the one 'A' in the middle with 'B' and form "AABBBBA".
The substring "BBBB" has the longest repeating letters, which is 4.
There may exists other ways to achieve this answer too.
```

 

**Constraints:**

- `1 <= s.length <= 10^5`
- `s` consists of only uppercase English letters.
- `0 <= k <= s.length`

## Solution

```python
class Solution:
    def characterReplacement(self, s: str, k: int) -> int:
        count = {}
        max_count = 0
        start = 0
        result = 0
        for end in range(len(s)):
            count[s[end]] = count.get(s[end], 0) + 1
            max_count = max(max_count, count[s[end]])
            if end - start + 1 - max_count > k:
                count[s[start]] -= 1
                start += 1
            result = max(result, end - start + 1)
        return result
```


## Explanation

### Approach: Sliding Window with Frequency Tracking

This solution uses a **sliding window** combined with a **frequency map** to find the longest substring with at most k character replacements.

### Intuition

**Sliding Window Logic:** Maintain a window where `window_length - max_frequency ≤ k` (needed replacements).

If the condition breaks, shrink from the left. The window always contains a valid substring.

**Track:** Frequency of each character in current window.

### Algorithm

1. Expand window by moving right pointer
2. Track character frequencies
3. If `needed replacements > k`, shrink from left
4. Update maximum length

### Why This Works

- **Valid Windows:** We maintain the invariant that window is always valid
- **O(n) Time:** Each pointer moves once through string
- **Greedy:** Always keep window as large as possible
- **Frequency Count:** O(26) with constant size map for letters


## Complexity

- **Time**: `O(n)`
- **Space**: `O(1)`
