## Problem Statement

Given two strings `ransomNote` and `magazine`, return `true`_ if _`ransomNote`_ can be constructed by using the letters from _`magazine`_ and _`false`_ otherwise_.

Each letter in `magazine` can only be used once in `ransomNote`.

 

**Example 1:**

```
Input: ransomNote = "a", magazine = "b"
Output: false
```
**Example 2:**

```
Input: ransomNote = "aa", magazine = "ab"
Output: false
```
**Example 3:**

```
Input: ransomNote = "aa", magazine = "aab"
Output: true
```

 

**Constraints:**

- `1 <= ransomNote.length, magazine.length <= 10^5`
- `ransomNote` and `magazine` consist of lowercase English letters.

## Solution

```python
from collections import Counter

class Solution:
    def canConstruct(self, ransomNote: str, magazine: str) -> bool:
        return not Counter(ransomNote) - Counter(magazine)
```


## Explanation

### Approach: Hash Map Lookup

This solution uses a **hash map** to store mappings and achieve O(1) lookup time for values or conversions.

### Intuition

The ransom note is constructible only if the magazine contains each required character at least as many times. Counting both strings gives us those frequencies directly, and subtracting the magazine counts from the ransom note counts reveals whether anything is still missing.

### Algorithm

Here's the step-by-step approach:

1. Build a `Counter` for `ransomNote`.
2. Build a `Counter` for `magazine`.
3. Subtract the magazine counts from the ransom-note counts.
4. Return whether the subtraction leaves no positive counts behind.

### Code Breakdown

`Counter(ransomNote) - Counter(magazine)` keeps only characters whose required count is still positive after using the magazine letters. If that result is empty, the magazine covers every needed character

### Why This Works

- **Frequency matching:** A character can be used only as many times as it appears in `magazine`.
- **Counter subtraction semantics:** Any leftover entry means the ransom note needs more of that character than the magazine provides.
- **Direct decision:** An empty remainder is exactly the condition for successful construction.



## Complexity

- **Time**: `O(m+n)`
- **Space**: `O(1)`
