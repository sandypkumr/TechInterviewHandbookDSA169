## Problem Statement

You are given an integer n representing the number of nodes labeled from 0 to n-1 and a list of edges where each edge
is a pair of nodes [u, v] that represents an undirected edge between nodes u and v.

Write a function to return the number of connected components in the graph.


## Examples

Example 1:

Example 1:
Input: n = 5, edges = [[0,1],[1,2],[3,4]]
Output: 2

Example 2:

Example 2:
Input: n = 5, edges = [[0,1],[1,2],[2,3],[3,4]]
Output: 1


## Solution

```python
from typing import List


class Solution:
    def countComponents(self, n: int, edges: List[List[int]]) -> int:
        graph = {i: [] for i in range(n)}
        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)
        visited = set()
        def dfs(node):
            visited.add(node)
            for neighbor in graph[node]:
                if neighbor not in visited:
                    dfs(neighbor)
        count = 0
        for i in range(n):
            if i not in visited:
                count += 1
                dfs(i)
        return count
```


## Explanation

### Approach: DFS per Unvisited Node

Build an adjacency list for the undirected graph, then:
- iterate all nodes
- if node not visited, it starts a new connected component
- run DFS to mark its entire component visited

Count how many DFS starts happen.

### Code Structure

```python
graph = {i: [] for i in range(n)}
for u, v in edges:
    graph[u].append(v)
    graph[v].append(u)

count = 0

        count += 1
        dfs(i)
```

DFS marks all nodes reachable from the start node.

### Why This Works

- Each DFS call covers exactly one connected component
- No node belongs to two components, so count is exact
- Every node and edge is processed at most once -> O(V+E)
