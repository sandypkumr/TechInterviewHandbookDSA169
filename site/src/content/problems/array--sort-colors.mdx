## Problem Statement

Given an array `nums` with `n` objects colored red, white, or blue, sort them **in-place **so that objects of the same color are adjacent, with the colors in the order red, white, and blue.

We will use the integers `0`, `1`, and `2` to represent the color red, white, and blue, respectively.

You must solve this problem without using the library's sort function.

 

**Example 1:**

```
Input: nums = [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]
```

**Example 2:**

```
Input: nums = [2,0,1]
Output: [0,1,2]
```

 

**Constraints:**

- `n == nums.length`
- `1 <= n <= 300`
- `nums[i]` is either `0`, `1`, or `2`.

 

**Follow up:** Could you come up with a one-pass algorithm using only constant extra space?

## Solution

```python
from typing import List


class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        left, right = 0, len(nums) - 1
        i = 0
        while i <= right:
            if nums[i] == 0:
                nums[left], nums[i] = nums[i], nums[left]
                left += 1
                i += 1
            elif nums[i] == 2:
                nums[right], nums[i] = nums[i], nums[right]
                right -= 1
            else:
                i += 1
```


## Explanation

### Approach: Dutch National Flag (Three Pointers)

Maintain three regions:
- `[0 .. left-1]` are all `0`
- `[left .. i-1]` are all `1`
- `[right+1 .. end]` are all `2`

Process index `i` while `i <= right`:
- if `nums[i] == 0`: swap with `left`, increment both `left` and `i`
- if `nums[i] == 2`: swap with `right`, decrement `right` only
- if `nums[i] == 1`: just increment `i`

### Why `i` Is Not Incremented After Swapping with `right`

The element swapped in from `right` has not been processed yet, so it must be checked in the next iteration.

### Code Core

```python
while i <= right:
    if nums[i] == 0:
        nums[left], nums[i] = nums[i], nums[left]
        left += 1
        i += 1
    elif nums[i] == 2:
        nums[right], nums[i] = nums[i], nums[right]
        right -= 1
    else:
        i += 1
```

### Why This Works

- Maintains the three-region invariant throughout
- Each element moves to its correct region in O(1) operations
- Single pass, O(n) time and O(1) extra space
