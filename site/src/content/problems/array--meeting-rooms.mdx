## Problem Statement

The "Meeting Rooms" problem is about determining if a person can attend all their meetings without overlap. Given an
array of intervals where each interval represents a meeting time [start, end], the goal is to check if any of these
intervals overlap. If there’s overlap, return False; otherwise, return True.


## Examples

For example:

Input: [[0, 30], [5, 10], [15, 20]]
Output: False (since the meetings overlap).


## Solution

```python
from typing import List

class Interval(object):
    def __init__(self, start, end):
        self.start = start
        self.end = end


class Solution:
    def canAttendMeetings(self, intervals: List[Interval]) -> bool:
        intervals.sort(key=lambda x: x.start)
        for i in range(1, len(intervals)):
            if intervals[i].start < intervals[i - 1].end:
                return False
        return True
```


## Explanation

### Approach: Sort by Start and Check Adjacent Overlaps

If meetings are sorted by start time, any conflict must appear between neighboring intervals in sorted order.

### Intuition

After sorting, if `intervals[i].start < intervals[i-1].end`, those two overlap, so one person cannot attend both.

### Algorithm

1. Sort intervals by start time
2. Scan from second interval onward
3. If any interval starts before previous one ends, return `False`
4. If no overlap found, return `True`

### Code Breakdown

```python
intervals.sort(key=lambda x: x.start)
for i in range(1, len(intervals)):
    if intervals[i].start < intervals[i - 1].end:

```

### Why This Works

- Sorting creates chronological order of starts.
- Any overlap must involve some adjacent pair in this order.
- One linear pass after sort is sufficient.


## Complexity

- **Time**: `O(n log n)`
- **Space**: `O(1)`
