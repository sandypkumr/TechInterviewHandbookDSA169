## Problem Statement

Given two strings `s` and `t`, return `true` _if they are equal when both are typed into empty text editors_. `'#'` means a backspace character.

Note that after backspacing an empty text, the text will continue empty.

 

**Example 1:**

```
Input: s = "ab#c", t = "ad#c"
Output: true
Explanation: Both s and t become "ac".
```

**Example 2:**

```
Input: s = "ab##", t = "c#d#"
Output: true
Explanation: Both s and t become "".
```

**Example 3:**

```
Input: s = "a#c", t = "b"
Output: false
Explanation: s becomes "c" while t becomes "b".
```

 

**Constraints:**

- `1 <= s.length, t.length <= 200`
- `s` and `t` only contain lowercase letters and `'#'` characters.

 

**Follow up:** Can you solve it in `O(n)` time and `O(1)` space?

## Solution

```python
class Solution:
    def backspaceCompare(self, s: str, t: str) -> bool:
        def build(s: str) -> str:
            stack = []
            for c in s:
                if c == '#':
                    if stack:
                        stack.pop()
                else:
                    stack.append(c)
            return ''.join(stack)
        return build(s) == build(t)
```


## Explanation

### Approach: Simulate Typing with a Stack

Process each string as if typing into an editor:
- normal character -> push to stack
- `#` -> pop last character if stack is not empty

After simulation, the stack content is the final typed text.
Run this for both strings and compare.

### Step-by-Step Example

`s = "ab#c"`
- `a` -> `[a]`
- `b` -> `[a,b]`
- `#` -> pop -> `[a]`
- `c` -> `[a,c]` -> final `"ac"`

`t = "ad#c"`
- `a` -> `[a]`
- `d` -> `[a,d]`
- `#` -> pop -> `[a]`
- `c` -> `[a,c]` -> final `"ac"`

Both equal -> `True`.

- Handles extra backspaces safely with `if stack`
- Each character processed once -> O(n) time
