## Problem Statement

Design a time-based key-value data structure that can store multiple values for the same key at different time stamps and retrieve the key's value at a certain timestamp.

Implement the `TimeMap` class:

- `TimeMap()` Initializes the object of the data structure.
- `void set(String key, String value, int timestamp)` Stores the key `key` with the value `value` at the given time `timestamp`.
- `String get(String key, int timestamp)` Returns a value such that `set` was called previously, with `timestamp_prev <= timestamp`. If there are multiple such values, it returns the value associated with the largest `timestamp_prev`. If there are no values, it returns `""`.

 

**Example 1:**

```
Input
["TimeMap", "set", "get", "get", "set", "get", "get"]
[[], ["foo", "bar", 1], ["foo", 1], ["foo", 3], ["foo", "bar2", 4], ["foo", 4], ["foo", 5]]
Output
[null, null, "bar", "bar", null, "bar2", "bar2"]

Explanation
TimeMap timeMap = new TimeMap();
timeMap.set("foo", "bar", 1); // store the key "foo" and value "bar" along with timestamp = 1.
timeMap.get("foo", 1); // return "bar"
timeMap.get("foo", 3); // return "bar", since there is no value corresponding to foo at timestamp 3 and timestamp 2, then the only value is at timestamp 1 is "bar".
timeMap.set("foo", "bar2", 4); // store the key "foo" and value "bar2" along with timestamp = 4.
timeMap.get("foo", 4); // return "bar2"
timeMap.get("foo", 5); // return "bar2"
```

 

**Constraints:**

- `1 <= key.length, value.length <= 100`
- `key` and `value` consist of lowercase English letters and digits.
- `1 <= timestamp <= 10^7`
- All the timestamps `timestamp` of `set` are strictly increasing.
- At most `2 * 10^5` calls will be made to `set` and `get`.

## Solution

```python
from collections import defaultdict


class TimeMap:

        def __init__(self):
            """
            Initialize your data structure here.
            """
            self.map = defaultdict(list)

        def set(self, key: str, value: str, timestamp: int) -> None:
            self.map[key].append((timestamp, value))

        def get(self, key: str, timestamp: int) -> str:
            if key not in self.map:
                return ""
            arr = self.map[key]
            left, right = 0, len(arr) - 1
            while left < right:
                mid = (left + right + 1) // 2
                if arr[mid][0] <= timestamp:
                    left = mid
                else:
                    right = mid - 1
            return arr[left][1] if arr[left][0] <= timestamp else ""
```


## Explanation

### Approach

This solution uses **Depth-First Search (DFS)** to explore all possible paths. It uses the **Two Pointers** technique to efficiently search through the data. It applies **Binary Search** to find the target in logarithmic time. It leverages a **Hash Map** for O(1) lookups and efficient tracking. It uses a **Set** for fast membership testing and duplicate detection.

### Intuition

By maintaining a sorted search space, we can eliminate half of the remaining candidates with each comparison, leading to logarithmic time complexity.

### Algorithm

1. **Handle edge cases**: Check for empty or invalid input
2. **Main loop**: Continue until condition is met
3. **Move pointers**: Adjust left/right pointers based on current sum
4. **Update hash map**: Store elements for future lookups
5. **Return result**: Output the computed answer

### Why This Works

- **Hash map efficiency**: O(1) average-case lookups dramatically reduce time complexity
- **Two pointers optimization**: Eliminates need for nested loops by intelligently moving pointers
- **Logarithmic search**: Halves search space with each iteration
- **Correctness**: The algorithm handles all edge cases and produces the expected output
- **Efficiency**: Optimized for both time and space constraints


## Complexity

- **Time**: `O(log n) per get`
- **Space**: `O(n)`
