## Problem Statement

Given an array of integers `nums` and an integer `k`, return _the total number of subarrays whose sum equals to_ `k`.

A subarray is a contiguous **non-empty** sequence of elements within an array.

 

**Example 1:**

```
Input: nums = [1,1,1], k = 2
Output: 2
```
**Example 2:**

```
Input: nums = [1,2,3], k = 3
Output: 2
```

 

**Constraints:**

- `1 <= nums.length <= 2 * 10^4`
- `-1000 <= nums[i] <= 1000`
- `-10^7 <= k <= 10^7`

## Solution

```python
from typing import List


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        count = 0
        sum_map = {0: 1}
        total = 0
        for num in nums:
            total += num
            count += sum_map.get(total - k, 0)
            sum_map[total] = sum_map.get(total, 0) + 1
        return count
```


## Explanation

### Approach: Prefix Sum + Hash Map

This solution combines **prefix sums** with a **hash map** to efficiently count subarrays with a specific sum. Instead of checking all subarrays (O(n²)), we use the prefix sum concept to reduce it to O(n).

### Intuition

**The Key Insight:** If two prefix sums differ by exactly `k`, then the subarray between them sums to `k`.

For example, if `prefix_sum[j] - prefix_sum[i] = k`, then the sum of elements from index `i+1` to `j` equals `k`.

**Algorithm idea:** As we iterate, maintain a hash map of prefix sums we've seen. For each position, check if `(current_prefix_sum - k)` exists in our map. If it does, we've found a valid subarray!

### Step-by-Step Walkthrough

Example: `nums = [1,1,1], k = 2`

Start with:
- `sum_map = {0: 1}`
- `total = 0`
- `count = 0`

```
i=0, num=1:
  total = 1
  need total-k = -1, not in map -> +0
  count = 0
  update map: {0:1, 1:1}

i=1, num=1:
  total = 2
  need total-k = 0, map[0]=1 -> +1
  count = 1
  update map: {0:1, 1:1, 2:1}

i=2, num=1:
  total = 3
  need total-k = 1, map[1]=1 -> +1
  count = 2
  update map: {0:1, 1:1, 2:1, 3:1}
```

Final answer: `count = 2` (subarrays `[1,1]` at indices `[0,1]` and `[1,2]`).

### Code Breakdown

```python
sum_map = {0: 1}
```
**Initialize with `{0: 1}`:** This handles the edge case where the prefix sum itself equals `k`.

```python
total += num
count += sum_map.get(total - k, 0)
```
**Core logic:** Count how many times we've seen `(total - k)`. Each occurrence represents a subarray ending here that sums to `k`.

```python
sum_map[total] = sum_map.get(total, 0) + 1
```
**Update map:** Store the current prefix sum for future lookups.

### Why This Works

- **Prefix Sum Property:** Subarray sum = prefix[j] - prefix[i]
- **O(n) Time:** Single pass with O(1) hash map operations
- **Space-Time Tradeoff:** Use O(n) space to achieve O(n) time instead of O(n²)
- **Handles Duplicates:** The map stores counts, so multiple subarrays with same prefix sum are counted correctly
