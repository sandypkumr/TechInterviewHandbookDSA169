## Problem Statement

Given the `root` of a binary tree, _check whether it is a mirror of itself_ (i.e., symmetric around its center).

 

**Example 1:**

```
Input: root = [1,2,2,3,4,4,3]
Output: true
```

**Example 2:**

```
Input: root = [1,2,2,null,3,null,3]
Output: false
```

 

**Constraints:**

- The number of nodes in the tree is in the range `[1, 1000]`.
- `-100 <= Node.val <= 100`

 

**Follow up:** Could you solve it both recursively and iteratively?

## Solution

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        def check(p, q):
            if p is None and q is None:
                return True
            if p is None or q is None:
                return False
            if p.val != q.val:
                return False
            return check(p.left, q.right) and check(p.right, q.left)
        return check(root, root)
```


## Explanation

### Approach

This solution uses **Depth-First Search (DFS)** to explore all possible paths. It uses the **Two Pointers** technique to efficiently search through the data.

### Intuition

The solution breaks down the problem into manageable steps, processing elements systematically to build the final answer.

### Algorithm

1. **Move pointers**: Adjust left/right pointers based on current sum
2. **Return result**: Output the computed answer

### Why This Works

- **Two pointers optimization**: Eliminates need for nested loops by intelligently moving pointers
- **Correctness**: The algorithm handles all edge cases and produces the expected output
- **Efficiency**: Optimized for both time and space constraints

- **Time**: `O(n)`
- **Space**: `O(h)`
