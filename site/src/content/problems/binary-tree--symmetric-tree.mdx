## Problem Statement

Given the `root` of a binary tree, _check whether it is a mirror of itself_ (i.e., symmetric around its center).

 

**Example 1:**

```
Input: root = [1,2,2,3,4,4,3]
Output: true
```

**Example 2:**

```
Input: root = [1,2,2,null,3,null,3]
Output: false
```

 

**Constraints:**

- The number of nodes in the tree is in the range `[1, 1000]`.
- `-100 <= Node.val <= 100`

 

**Follow up:** Could you solve it both recursively and iteratively?

## Solution

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        def check(p, q):
            if p is None and q is None:
                return True
            if p is None or q is None:
                return False
            if p.val != q.val:
                return False
            return check(p.left, q.right) and check(p.right, q.left)
        return check(root, root)
```


## Explanation

### Approach: Mirror DFS (Pairwise Comparison)

A tree is symmetric if its left and right subtrees are mirror images.

Mirror condition for two nodes `p` and `q`:
1. both null -> symmetric at this branch
2. one null -> not symmetric
3. values equal
4. `p.left` mirrors `q.right` and `p.right` mirrors `q.left`

### Code Logic

```python
if p is None and q is None:
    return True
if p is None or q is None:
    return False
if p.val != q.val:
    return False
return check(p.left, q.right) and check(p.right, q.left)
```

Initial call is `check(root, root)` so each side is compared with its mirror counterpart.

### Step-by-Step Intuition

For `[1,2,2,3,4,4,3]`:
- compare `(2,2)` -> equal
- compare `(3,3)` and `(4,4)` in mirrored positions -> equal
- all mirrored pairs match -> symmetric

### Why This Works

- Recursion enforces mirror relation at every depth
- Any structural/value mismatch is caught immediately
- Each node participates in constant-time checks -> O(n) time


## Complexity

- **Time**: `O(n)`
- **Space**: `O(h)`
