## Problem Statement

Given a string `s` and a dictionary of strings `wordDict`, return `true` if `s` can be segmented into a space-separated sequence of one or more dictionary words.

**Note** that the same word in the dictionary may be reused multiple times in the segmentation.

 

**Example 1:**

```
Input: s = "leetcode", wordDict = ["leet","code"]
Output: true
Explanation: Return true because "leetcode" can be segmented as "leet code".
```

**Example 2:**

```
Input: s = "applepenapple", wordDict = ["apple","pen"]
Output: true
Explanation: Return true because "applepenapple" can be segmented as "apple pen apple".
Note that you are allowed to reuse a dictionary word.
```

**Example 3:**

```
Input: s = "catsandog", wordDict = ["cats","dog","sand","and","cat"]
Output: false
```

 

**Constraints:**

- `1 <= s.length <= 300`
- `1 <= wordDict.length <= 1000`
- `1 <= wordDict[i].length <= 20`
- `s` and `wordDict[i]` consist of only lowercase English letters.
- All the strings of `wordDict` are **unique**.

## Solution

```python
from typing import List


class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        word_set = set(wordDict)
        n = len(s)
        dp = [False] * (n + 1)
        dp[0] = True
        for i in range(1, n + 1):
            for j in range(i):
                if dp[j] and s[j:i] in word_set:
                    dp[i] = True
                    break
        return dp[n]
```


## Explanation

### Approach: Dynamic Programming

This solution uses **Dynamic Programming** to build answers from smaller subproblems and avoid recomputation.

### Intuition

Let `dp[i]` mean whether the prefix `s[:i]` can be segmented into words from the dictionary. If there is some split point `j < i` such that `dp[j]` is true and `s[j:i]` is a dictionary word, then the prefix up to `i` is also valid.

### Algorithm

Here's the step-by-step approach:

1. Convert `wordDict` into a set for fast membership checks.
2. Create a boolean DP array of length `n + 1` and set `dp[0] = True` for the empty prefix.
3. For each ending index `i`, try every earlier split point `j`. If `dp[j]` is true and `s[j:i]` is in the set, mark `dp[i] = True`.
4. Return `dp[n]` to indicate whether the whole string can be segmented.

### Code Breakdown

The outer loop grows the prefix length, and the inner loop searches for the first valid split point. The `break` stops early once `dp[i]` becomes true, which avoids unnecessary substring checks.

### Why This Works

- **Correct state meaning:** `dp[i]` is true exactly when `s[:i]` can be segmented.
- **Complete transitions:** Trying every split point guarantees that every possible last word is considered.
- **Bottom-up construction:** Smaller prefixes are solved before larger ones, so each decision uses already computed information.
- **Optimization:** Balances time complexity gains with reasonable space usage


## Complexity

- **Time**: `O(n²)`
- **Space**: `O(n)`
