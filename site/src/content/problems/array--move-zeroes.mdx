## Problem Statement

Given an integer array `nums`, move all `0`'s to the end of it while maintaining the relative order of the non-zero elements.

**Note** that you must do this in-place without making a copy of the array.

 

**Example 1:**

```
Input: nums = [0,1,0,3,12]
Output: [1,3,12,0,0]
```
**Example 2:**

```
Input: nums = [0]
Output: [0]
```

 

**Constraints:**

- `1 <= nums.length <= 10^4`
- `-2^31 <= nums[i] <= 2^31 - 1`

 

**Follow up:** Could you minimize the total number of operations done?

## Solution

```python
from typing import List


class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        non_zero_index = 0
        for i in range(len(nums)):
            if nums[i] != 0:
                nums[non_zero_index], nums[i] = nums[i], nums[non_zero_index]
                non_zero_index += 1
```


## Explanation

### Approach: Two Pointers with In-Place Swapping

Use two pointers: `non_zero_index` tracks where the next non-zero element should go, and `i` scans through the array. When we find a non-zero element, swap it into the `non_zero_index` position.

### Why This Works

We're not creating a new array or removing elements. Instead, we're rearranging elements in-place by swapping non-zero elements forward. This maintains relative order and uses O(1) space.

### Key Insight

**All zeros move to the end naturally.** As we move non-zero elements forward, zeros get pushed back to fill the remaining positions.

### Step-by-Step Example

**Input:** `nums = [0,1,0,3,12]`

```
Initial:              [0, 1, 0, 3, 12]  non_zero_index=0

i=0 (0): skip
i=1 (1): non-zero!
  Swap nums[0] and nums[1]  → [1, 0, 0, 3, 12]  non_zero_index=1

i=2 (0): skip
i=3 (3): non-zero!
  Swap nums[1] and nums[3]  → [1, 3, 0, 0, 12]  non_zero_index=2

i=4 (12): non-zero!
  Swap nums[2] and nums[4]  → [1, 3, 12, 0, 0]  non_zero_index=3

Result:   [1, 3, 12, 0, 0] ✓ (relative order of non-zeros preserved!)
```

### The Code

```python
non_zero_index = 0
for i in range(len(nums)):
    if nums[i] != 0:
        nums[non_zero_index], nums[i] = nums[i], nums[non_zero_index]
        non_zero_index += 1
```

**How it works:**
- `non_zero_index` points to where the next non-zero should be placed
- When we find a non-zero at position `i`, we swap it there
- Then increment `non_zero_index` for the next non-zero

### Why O(1) Space is Possible

We're modifying in-place - no extra data structures needed. The swap itself is just two assignments.


## Complexity

- **Time**: `O(n)`
- **Space**: `O(1)`
