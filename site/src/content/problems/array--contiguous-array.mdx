## Problem Statement

Given a binary array `nums`, return _the maximum length of a contiguous subarray with an equal number of _`0`_ and _`1`.

 

**Example 1:**

```
Input: nums = [0,1]
Output: 2
Explanation: [0, 1] is the longest contiguous subarray with an equal number of 0 and 1.
```

**Example 2:**

```
Input: nums = [0,1,0]
Output: 2
Explanation: [0, 1] (or [1, 0]) is a longest contiguous subarray with equal number of 0 and 1.
```

**Example 3:**

```
Input: nums = [0,1,1,1,1,1,0,0,0]
Output: 6
Explanation: [1,1,1,0,0,0] is the longest contiguous subarray with equal number of 0 and 1.
```

 

**Constraints:**

- `1 <= nums.length <= 10^5`
- `nums[i]` is either `0` or `1`.

## Solution

```python
from typing import List


class Solution:
    def findMaxLength(self, nums: List[int]) -> int:
        count = 0
        max_length = 0
        count_map = {0: -1}
        for i, num in enumerate(nums):
            count += 1 if num == 1 else -1
            if count in count_map:
                max_length = max(max_length, i - count_map[count])
            else:
                count_map[count] = i
        return max_length
```


## Explanation

### Approach: Running Balance + First Occurrence Hash Map

Treat `1` as `+1` and `0` as `-1`. Then the problem becomes:
find the longest subarray with sum `0`.

If the same running balance appears at two indices, the subarray between them has net sum 0, which means equal numbers of 0s and 1s.

### Key Insight

`balance[i] == balance[j]` (with `i < j`) means:
- sum from `i+1` to `j` is 0
- so that range contains equal count of 0 and 1

So we store the **first index** where each balance is seen.

### Step-by-Step Walkthrough

**Input:** `nums = [0,1,0]`

- Start: `count = 0`, `count_map = {0: -1}`, `max_length = 0`

1. `i=0`, num=0 -> `count = -1`
   - first time seeing `-1`, store `count_map[-1] = 0`

2. `i=1`, num=1 -> `count = 0`
   - `0` seen before at index `-1`
   - length = `1 - (-1) = 2` -> `max_length = 2`

3. `i=2`, num=0 -> `count = -1`
   - `-1` seen before at index `0`
   - length = `2 - 0 = 2` (max stays 2)

Answer = 2

### Code Breakdown

```python
count += 1 if num == 1 else -1
if count in count_map:
    max_length = max(max_length, i - count_map[count])
else:
    count_map[count] = i
```

- Update running balance
- If balance seen before, compute zero-sum window length
- If not seen, store first occurrence only (critical for max length)

### Why This Works

- Converts equal-0/1 condition into zero-sum condition
- Hash map gives O(1) access to earliest index per balance
- Single pass over array gives O(n) time
