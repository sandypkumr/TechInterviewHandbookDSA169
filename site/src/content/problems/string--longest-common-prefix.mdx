## Problem Statement

Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string `""`.

 

**Example 1:**

```
Input: strs = ["flower","flow","flight"]
Output: "fl"
```

**Example 2:**

```
Input: strs = ["dog","racecar","car"]
Output: ""
Explanation: There is no common prefix among the input strings.
```

 

**Constraints:**

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` consists of only lowercase English letters if it is non-empty.

## Solution

```python
from typing import List


class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        if not strs:
            return ''
        for i, chars in enumerate(zip(*strs)):
            if len(set(chars)) > 1:
                return strs[0][:i]
        return min(strs)
```


## Explanation

### Approach: Vertical Scan with `zip`

Compare all strings character-by-character at the same index. Python's `zip(*strs)` groups the first characters together, then the second characters, and so on. As soon as one column contains different letters, the common prefix ends.

### Intuition

The common prefix can only extend while **every** string agrees at the current position. The first mismatch immediately tells us where to stop.

### Algorithm

1. If the input list is empty, return `""`
2. Iterate through grouped character columns using `zip(*strs)`
3. For each column, check whether all characters are the same
4. If a mismatch appears at index `i`, return `strs[0][:i]`
5. If every scanned column matches, the shortest string is the answer

### Code Breakdown

- `zip(*strs)`: groups together characters at the same position across all strings
- `len(set(chars)) > 1`: detects the first mismatch in a column
- `return strs[0][:i]`: returns the prefix before the mismatch
- `return min(strs)`: if no mismatch appears, the shortest string is the full common prefix

### Why This Works

- A prefix is common only if every string matches in every prefix position
- The first mismatching column is exactly where the longest common prefix ends
- If no mismatch is found, only the shortest string can be the answer because `zip` stops there


## Complexity

- **Time**: `O(n k)`
- **Space**: `O(1)`
