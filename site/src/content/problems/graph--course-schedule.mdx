## Problem Statement

There are a total of `numCourses` courses you have to take, labeled from `0` to `numCourses - 1`. You are given an array `prerequisites` where `prerequisites[i] = [a_i, b_i]` indicates that you **must** take course `b_i` first if you want to take course `a_i`.

- For example, the pair `[0, 1]`, indicates that to take course `0` you have to first take course `1`.

Return `true` if you can finish all courses. Otherwise, return `false`.

 

**Example 1:**

```
Input: numCourses = 2, prerequisites = [[1,0]]
Output: true
Explanation: There are a total of 2 courses to take. 
To take course 1 you should have finished course 0. So it is possible.
```

**Example 2:**

```
Input: numCourses = 2, prerequisites = [[1,0],[0,1]]
Output: false
Explanation: There are a total of 2 courses to take. 
To take course 1 you should have finished course 0, and to take course 0 you should also have finished course 1. So it is impossible.
```

 

**Constraints:**

- `1 <= numCourses <= 2000`
- `0 <= prerequisites.length <= 5000`
- `prerequisites[i].length == 2`
- `0 <= a_i, b_i < numCourses`
- All the pairs prerequisites[i] are **unique**.

## Solution

```python
from typing import List


class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        graph = {i: [] for i in range(numCourses)}
        for course, pre in prerequisites:
            graph[course].append(pre)
        visited = [0] * numCourses
        for i in range(numCourses):
            if not self.dfs(graph, visited, i):
                return False
        return True

    def dfs(self, graph, visited, i):
        if visited[i] == -1:
            return False
        if visited[i] == 1:
            return True
        visited[i] = -1
        for j in graph[i]:
            if not self.dfs(graph, visited, j):
                return False
        visited[i] = 1
        return True
```


## Explanation

### Approach: DFS Cycle Detection in Directed Graph

Model courses as a directed graph:
- edge `course -> prereq`

You can finish all courses **iff** graph has no cycle.

This DFS uses 3 states in `visited`:
- `0` = unvisited
- `-1` = currently in recursion stack (visiting)
- `1` = fully processed (safe)

If DFS reaches a node already marked `-1`, we found a back-edge -> cycle exists.

### Step-by-Step Intuition

For prerequisites `[[1,0],[0,1]]`:

- DFS(0): mark `0` as visiting (`-1`)
- Explore prereq `1`: DFS(1), mark `1` as visiting
- Explore prereq `0`: sees `visited[0] == -1` -> cycle -> cannot finish

Return `False`.

### Code Breakdown

```python
if visited[i] == -1:
    return False    # cycle
if visited[i] == 1:
    return True     # already verified acyclic

    if not self.dfs(graph, visited, j):
        return False

```

Outer loop runs DFS from every course to cover disconnected components.

### Why This Works

- Detects exactly the condition that blocks completion: directed cycle
- Each node and edge is processed at most once in successful traversal
- Complexity O(V+E)
