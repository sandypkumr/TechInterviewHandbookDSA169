## Problem Statement

Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

 

**Example 1:**

**Input:** s = "anagram", t = "nagaram"

**Output:** true

**Example 2:**

**Input:** s = "rat", t = "car"

**Output:** false

 

**Constraints:**

- `1 <= s.length, t.length <= 5 * 10^4`
- `s` and `t` consist of lowercase English letters.

 

**Follow up:** What if the inputs contain Unicode characters? How would you adapt your solution to such a case?

## Solution

```python
from collections import Counter


class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        return Counter(s) == Counter(t)
```


## Explanation

### Approach: Frequency Count Comparison with `Counter`

Two strings are anagrams if and only if every character appears the same number of times in both strings.

`Counter` builds frequency maps, so equality check directly answers the problem.

### Step-by-Step Example

`s = "anagram"`, `t = "nagaram"`

- `Counter(s)` -> `{a:3, n:1, g:1, r:1, m:1}`
- `Counter(t)` -> `{a:3, n:1, g:1, r:1, m:1}`

Equal -> anagram (`True`).

`s = "rat"`, `t = "car"`

- `Counter(s)` has `t:1`
- `Counter(t)` has `c:1`

Not equal -> not anagram (`False`).

### Code Breakdown

```python
return Counter(s) == Counter(t)
```

- Build two frequency dictionaries
- Compare key-value pairs

### Complexity Note

For fixed lowercase English alphabet, auxiliary space is bounded by alphabet size, often treated as O(1).
For general Unicode, distinct characters can grow with input, so space is O(k) where `k` is number of unique chars.


## Complexity

- **Time**: `O(n)`
- **Space**: `O(1)`
