## Problem Statement

Determine if a `9 x 9` Sudoku board is valid. Only the filled cells need to be validated **according to the following rules**:

1. Each row must contain the digits `1-9` without repetition.
2. Each column must contain the digits `1-9` without repetition.
3. Each of the nine `3 x 3` sub-boxes of the grid must contain the digits `1-9` without repetition.

**Note:**

- A Sudoku board (partially filled) could be valid but is not necessarily solvable.
- Only the filled cells need to be validated according to the mentioned rules.

 

**Example 1:**

```
Input: board = 
[["5","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
Output: true
```

**Example 2:**

```
Input: board = 
[["8","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
Output: false
Explanation: Same as Example 1, except with the 5 in the top left corner being modified to 8. Since there are two 8's in the top left 3x3 sub-box, it is invalid.
```

 

**Constraints:**

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` is a digit `1-9` or `'.'`.

## Solution

```python
from typing import List


class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        rows = [set() for _ in range(9)]
        cols = [set() for _ in range(9)]
        boxes = [set() for _ in range(9)]
        for row in range(9):
            for col in range(9):
                if board[row][col] == '.':
                    continue
                box = (row // 3) * 3 + col // 3
                if board[row][col] in rows[row] or board[row][col] in cols[col] or board[row][col] in boxes[box]:
                    return False
                rows[row].add(board[row][col])
                cols[col].add(board[row][col])
                boxes[box].add(board[row][col])
        return True
```


## Explanation

### Approach: Track Seen Digits in Rows, Columns, and Boxes

As we scan the board, each filled digit must be unique in three places at once:

1. its row
2. its column
3. its `3 x 3` sub-box

We keep three arrays of sets to record what has already appeared. If a digit is already present in any of those sets, the board is invalid.

### Intuition

Sudoku validity is entirely local: a digit only conflicts with other digits in the same row, column, or box. That makes sets a natural fit for constant-time duplicate checks while scanning the grid exactly once.

### Algorithm

1. Create 9 sets for rows, 9 sets for columns, and 9 sets for boxes
2. Visit every cell in the board
3. Skip `'.'` because empty cells do not matter
4. Compute the box index with `(row // 3) * 3 + col // 3`
5. If the digit is already in the matching row, column, or box set, return `False`
6. Otherwise insert it into all three sets
7. If the whole board is processed without conflict, return `True`

### Code Breakdown

- `rows = [set() for _ in range(9)]`: digits already seen in each row
- `cols = [set() for _ in range(9)]`: digits already seen in each column
- `boxes = [set() for _ in range(9)]`: digits already seen in each `3 x 3` box
- `box = (row // 3) * 3 + col // 3`: maps a cell to box `0..8`
- `if board[row][col] in rows[row] ...`: duplicate check before insertion

### Why This Works

- Every filled cell is validated against exactly the three Sudoku rules
- A duplicate is detected immediately when it first becomes illegal
- Because the board size is fixed at `9 x 9`, the work is bounded and constant in practice


## Complexity

- **Time**: `O(1)`
- **Space**: `O(1)`
