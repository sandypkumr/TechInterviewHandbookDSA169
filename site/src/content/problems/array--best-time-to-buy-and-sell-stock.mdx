## Problem Statement

You are given an array `prices` where `prices[i]` is the price of a given stock on the `i^th` day.

You want to maximize your profit by choosing a **single day** to buy one stock and choosing a **different day in the future** to sell that stock.

Return _the maximum profit you can achieve from this transaction_. If you cannot achieve any profit, return `0`.

 

**Example 1:**

```
Input: prices = [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.
Note that buying on day 2 and selling on day 1 is not allowed because you must buy before you sell.
```

**Example 2:**

```
Input: prices = [7,6,4,3,1]
Output: 0
Explanation: In this case, no transactions are done and the max profit = 0.
```

 

**Constraints:**

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

## Solution

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float('inf')
        max_profit = 0
        for price in prices:
            min_price = min(min_price, price)
            max_profit = max(max_profit, price - min_price)
        return max_profit
```


## Explanation

### Approach: Greedy - Track Minimum Price and Maximum Profit

This is a **greedy algorithm** that makes a single pass through prices. At each day, we:
1. Track the minimum price seen so far
2. Calculate profit if we sold today at current price
3. Update the maximum profit

### Why Greedy Works Here

For stock trading on multiple days, we can sell on any day. So the optimal strategy is: find the lowest price before each day, then sell at that day's price. A greedy approach captures this perfectly.

### Step-by-Step Example

**Input:** `prices = [7,1,5,3,6,4]`

```
Day 0 (price=7):
  min_price = 7, profit = 0

Day 1 (price=1):
  profit_if_sold = 1 - 7 = -6
  min_price = 1 (update to lower value)

Day 2 (price=5):
  profit_if_sold = 5 - 1 = 4 ✓ (max profit so far)

Day 3 (price=3):
  profit_if_sold = 3 - 1 = 2

Day 4 (price=6):
  profit_if_sold = 6 - 1 = 5 ✓ (new max!)

Day 5 (price=4):
  profit_if_sold = 4 - 1 = 3

Result: 5 (buy at 1, sell at 6)
```

### Code

```python
min_price = float('inf')
max_profit = 0
for price in prices:
    profit = price - min_price
    max_profit = max(max_profit, profit)
    min_price = min(min_price, price)
```

Simple and elegant! Three operations per iteration.

### Why This Works

- **One Pass:** O(n) - we only look at each price once
- **No Backtracking:** We never need to revisit decisions
- **Always Optimal:** Buying at the lowest and selling at the highest after that is always optimal
