## Problem Statement

Given an array of strings `strs`, group the anagrams together. You can return the answer in **any order**.

 

**Example 1:**

**Input:** strs = ["eat","tea","tan","ate","nat","bat"]

**Output:** [["bat"],["nat","tan"],["ate","eat","tea"]]

**Explanation:**

- There is no string in strs that can be rearranged to form `"bat"`.
- The strings `"nat"` and `"tan"` are anagrams as they can be rearranged to form each other.
- The strings `"ate"`, `"eat"`, and `"tea"` are anagrams as they can be rearranged to form each other.

**Example 2:**

**Input:** strs = [""]

**Output:** [[""]]

**Example 3:**

**Input:** strs = ["a"]

**Output:** [["a"]]

 

**Constraints:**

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` consists of lowercase English letters.

## Solution

```python
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        anagrams = {}
        for s in strs:
            key = tuple(sorted(s))
            if key not in anagrams:
                anagrams[key] = []
            anagrams[key].append(s)
        return list(anagrams.values())
```


## Explanation

### Approach: Hash by Canonical Sorted Key

Two strings are anagrams if their sorted characters are identical.
So for each string, build a canonical key `tuple(sorted(s))`, and group strings by that key in a hash map.

### Step-by-Step Example

Input: `['eat','tea','tan','ate','nat','bat']`

- `eat` -> key `('a','e','t')`
- `tea` -> key `('a','e','t')`
- `ate` -> key `('a','e','t')`
  -> same bucket

- `tan`, `nat` -> key `('a','n','t')`
- `bat` -> key `('a','b','t')`

Return all bucket values.

### Code Breakdown

```python
anagrams = {}
```

Tuple is used as hashable dictionary key.

### Why This Works

- Canonical key ensures all and only anagrams map together
- Hash map provides O(1) average insertion per string
- Sorting dominates per-string cost: O(k log k)
