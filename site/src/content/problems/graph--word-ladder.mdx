## Problem Statement

A **transformation sequence** from word `beginWord` to word `endWord` using a dictionary `wordList` is a sequence of words `beginWord -> s_1 -> s_2 -> ... -> s_k` such that:

- Every adjacent pair of words differs by a single letter.
- Every `s_i` for `1 <= i <= k` is in `wordList`. Note that `beginWord` does not need to be in `wordList`.
- `s_k == endWord`

Given two words, `beginWord` and `endWord`, and a dictionary `wordList`, return _the **number of words** in the **shortest transformation sequence** from_ `beginWord` _to_ `endWord`_, or _`0`_ if no such sequence exists._

 

**Example 1:**

```
Input: beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log","cog"]
Output: 5
Explanation: One shortest transformation sequence is "hit" -> "hot" -> "dot" -> "dog" -> cog", which is 5 words long.
```

**Example 2:**

```
Input: beginWord = "hit", endWord = "cog", wordList = ["hot","dot","dog","lot","log"]
Output: 0
Explanation: The endWord "cog" is not in wordList, therefore there is no valid transformation sequence.
```

 

**Constraints:**

- `1 <= beginWord.length <= 10`
- `endWord.length == beginWord.length`
- `1 <= wordList.length <= 5000`
- `wordList[i].length == beginWord.length`
- `beginWord`, `endWord`, and `wordList[i]` consist of lowercase English letters.
- `beginWord != endWord`
- All the words in `wordList` are **unique**.

## Solution

```python
import string
from collections import deque
from typing import List


class Solution:
    def ladderLength(self, beginWord: str, endWord: str, wordList: List[str]) -> int:
        word_set = set(wordList)
        if endWord not in word_set:
            return 0
        queue = deque([(beginWord, 1)])
        while queue:
            word, steps = queue.popleft()
            if word == endWord:
                return steps
            for i in range(len(word)):
                for c in string.ascii_lowercase:
                    next_word = word[:i] + c + word[i + 1:]
                    if next_word in word_set:
                        word_set.remove(next_word)
                        queue.append((next_word, steps + 1))
        return 0
```


## Explanation

### Approach: BFS

This solution uses **Breadth-First Search (BFS)** with a queue to explore states level by level.

### Intuition

Think of each word as a node in a graph, with an edge between words that differ by one letter. Because each transformation has equal cost, BFS finds the shortest transformation sequence by exploring all words reachable in `d` steps before moving to `d + 1` steps.

### Algorithm

Here's the step-by-step approach:

1. Put all dictionary words into a set and return `0` immediately if `endWord` is missing.
2. Start BFS from `beginWord` with step count `1`.
3. For each word, try replacing every character with every lowercase letter to generate neighboring words.
4. If a generated word is in the set, remove it, enqueue it with step count `+1`, and return as soon as `endWord` is reached.

### Code Breakdown

The nested generation loop builds all one-letter mutations of the current word. Removing a discovered word from `word_set` doubles as a visited marker, ensuring each candidate is processed at most once.

### Why This Works

- **Graph interpretation:** Each valid one-letter change is an edge in the transformation graph.
- **BFS optimality:** The first time `endWord` is dequeued or generated, it is reached by the shortest sequence.
- **Visited pruning:** Removing used words prevents revisiting nodes and keeps the search finite.
- **Handles edge cases:** Correctly processes boundary conditions and special inputs
- **Time-space optimization:** Balances time complexity gains with reasonable space usage

- **Time**: `O(M²×N)`
- **Space**: `O(M²×N)`
