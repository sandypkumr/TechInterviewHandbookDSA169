## Core Idea

Reduce search space by half each iteration. **Requires sorted order or monotonic predicate.**

```
O(log n) time — O(1) space
```

---

## Template 1 — Basic (exact match)

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2   # avoids overflow
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

## Template 2 — Left Bound (first position where arr[i] >= target)

```python
def left_bound(arr, target):
    left, right = 0, len(arr)          # right is exclusive
    while left < right:
        mid = (left + right) // 2
        if arr[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left   # first index where arr[left] >= target
```

**Returns:** index of leftmost `target`, or insertion point if not found.

---

## Template 3 — Right Bound (last position where arr[i] \<= target)

```python
def right_bound(arr, target):
    left, right = 0, len(arr)
    while left < right:
        mid = (left + right) // 2
        if arr[mid] <= target:
            left = mid + 1
        else:
            right = mid
    return left - 1   # last index where arr[i] <= target
```

## Template 4 — Answer Space (bisect on the answer)

Use when: "find minimum X such that condition(X) is True".

```python
def answer_space(lo, hi, condition):
    """
    condition(x) is False for x < answer, True for x >= answer.
    Returns smallest x in [lo, hi] where condition(x) is True.
    """
    while lo < hi:
        mid = (lo + hi) // 2
        if condition(mid):
            hi = mid          # mid could be the answer — narrow right
        else:
            lo = mid + 1      # mid is too small — move right
    return lo
```

**Examples:**
- Koko Eating Bananas — condition: can eat all bananas at speed `mid` in `h` hours
- Capacity To Ship Packages — condition: can ship all packages with capacity `mid`
- Split Array Largest Sum — condition: can split into `m` parts each ≤ `mid`

## Python's `bisect` Module

```python
import bisect

bisect.bisect_left(arr, x)    # leftmost insertion point for x
bisect.bisect_right(arr, x)   # rightmost insertion point for x
bisect.insort_left(arr, x)    # insert x maintaining sorted order
bisect.insort_right(arr, x)

# Count elements equal to x in sorted arr:
count = bisect.bisect_right(arr, x) - bisect.bisect_left(arr, x)

# Find floor (largest element <= x):
i = bisect.bisect_right(arr, x) - 1
floor = arr[i] if i >= 0 else None

# Find ceil (smallest element >= x):
i = bisect.bisect_left(arr, x)
ceil = arr[i] if i < len(arr) else None
```

## Search in Rotated Array

```python
def search_rotated(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        if nums[left] <= nums[mid]:          # left half is sorted
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:                                # right half is sorted
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1
    return -1
```

## Decision Guide

| Condition | Template |
|---|---|
| Find exact value | Basic |
| First/last occurrence | Left/Right bound |
| Count occurrences | Right bound - Left bound |
| Min feasible answer | Answer space (condition = feasibility) |
| Max feasible answer | Answer space (negate condition) |
| Rotated sorted array | Special rotation check |
